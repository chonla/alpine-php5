# chonla/alpine-php

Docker image create php and [some components](#components) on top of alpine linux.

## Components

* git
* curl
* openssl
* php5
* php5-ctype
* php5-curl
* php5-iconv
* php5-json
* php5-mcrypt
* php5-openssl
* php5-pdo
* php5-pdo_sqlite
* php5-phar
* sqlite
